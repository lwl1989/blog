在PHP中字符数字在和整数比较时，可以看作进行了一次intval。
==============================================================================
```
<?php
var_dump('0'==0);
var_dump('1'==0);
var_dump("1"==0);
var_dump('a'==0);
?>
true    intval('0')=0
false   intval('1')=1
false   intval('1')=1 
true    intval('a')=0
```
